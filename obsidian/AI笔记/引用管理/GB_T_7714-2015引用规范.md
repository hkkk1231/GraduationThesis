# GB/T 7714-2015 引用格式指南

## 📋 标准概述

**GB/T 7714-2015《信息与文献 参考文献著录规则》**是中国现行的参考文献著录国家标准，于2015年12月1日实施，替代了GB/T 7714-2005标准。

## 🎯 在Zotero中配置

### 1. 安装引用样式
1. 下载 `china-national-standard-gb-t-7714-2015-numeric.csl`
2. Zotero → 编辑 → 首选项 → 引用 → "+" → 安装样式
3. 选择下载的CSL文件

### 2. 设置为默认样式
1. 在引用样式列表中找到"Chinese Std GB/T 7714-2015 (numeric)"
2. 双击设置为默认样式

### 3. 生成引用
1. 在Zotero中选择文献
2. 按 `Ctrl+Shift+C` 复制引用
3. 粘贴到文档中即可

## 📝 在Obsidian中使用

### 直接从Zotero复制
```
[1] 李荣琼. 基于人工智能技术的语文课堂教学设计与分析[J]. 未明确期刊, 2025(10): 146-147.
```

### 在笔记中引用
```markdown
## 正文引用
李荣琼(2025)的研究表明...

## 参考文献
[1] 李荣琼. 基于人工智能技术的语文课堂教学设计与分析[J]. 未明确期刊, 2025(10): 146-147.

## 链接到文献笔记
参见：[[李荣琼_2025_基于人工智能技术的语文课堂教学设计与分析]]
```

## 📚 常见文献格式

### 期刊论文
```
[1] 作者. 题名[J]. 期刊名, 年, 卷(期): 起止页码.
```

### 专著
```
[1] 作者. 书名[M]. 版本. 出版地: 出版者, 出版年.
```

### 学位论文
```
[1] 作者. 题名[D]. 保存地点: 保存单位, 年份.
```

## ⚡ 使用技巧

### 快速操作
- `Ctrl+Shift+C`: 复制引用
- `Ctrl+Shift+B`: 复制参考文献
- `Ctrl+Shift+P`: 复制带引号的引用

### 批量操作
1. 选择多篇文献
2. 按 `Ctrl+Shift+C` 生成多个引用
3. 按顺序粘贴到参考文献列表

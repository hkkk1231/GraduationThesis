## 📝 测试步骤

### 步骤1: 安装Zotfile插件
```cmd
E:\仓库\毕业论文\install_zotfile.bat
```

### 步骤2: 配置Zotfile设置
1. 打开Zotero
2. 工具 → Zotfile Preferences
3. 设置Tablet文件夹为: `E:\仓库\毕业论文\obsidian\AI笔记\PDF阅读`
4. 启用注释提取功能

### 步骤3: 安装Obsidian插件
1. 打开Obsidian
2. 设置 → 社区插件 → 浏览
3. 搜索并安装必需插件
4. 配置插件设置

### 步骤4: 测试同步功能
1. 在Zotero中选择文献
2. 右键 → Manage Attachments → Send to Tablet
3. 检查PDF是否移动到Obsidian文件夹
4. 检查注释是否提取为笔记

## 📊 元数据字段

### 基础元数据
```yaml
---
title: 文献标题
authors: 作者列表
publication: 发表期刊
year: 发表年份
volume: 卷号
issue: 期号
pages: 页码
doi: DOI号
url: 链接地址
tags: [标签列表]
citekey: 引用key
---
```

### 扩展元数据
```yaml
---
文献类型: 期刊文章/专著/学位论文/会议论文
学科分类: 教育技术学/计算机科学/心理学
研究对象: 具体研究对象
研究方法: 实验法/调查法/案例研究
技术基础: 相关技术基础
应用场景: 实际应用场景
关键词: 核心关键词
研究质量: ⭐⭐⭐⭐⭐ (1-5星)
阅读状态: 未读/已读/精读
重要程度: 低/中/高
应用价值: 低/中/高
创建时间: YYYY-MM-DD
更新时间: YYYY-MM-DD
---
```

## 🎯 质量评级标准

### 研究质量评级
- ⭐⭐⭐⭐⭐ (5星) - 顶级期刊，方法严谨，贡献重大
- ⭐⭐⭐⭐ (4星) - 优质期刊，方法规范，贡献明显
- ⭐⭐⭐ (3星) - 一般期刊，方法基本，贡献一般
- ⭐⭐ (2星) - 普通期刊，方法简单，贡献有限
- ⭐ (1星) - 质量较低，参考价值有限

### 重要程度评级
- **高** - 核心参考文献，必须精读
- **中** - 重要参考文献，建议阅读
- **低** - 一般参考文献，可选择性阅读

### 应用价值评级
- **高** - 直接相关，可立即应用
- **中** - 间接相关，有参考价值
- **低** - 理论相关，应用较少

## 🔍 Dataview查询示例

### 按年份查询文献
```dataview
TABLE 
  authors as "作者",
  publication as "期刊",
  研究质量 as "质量",
  重要程度 as "重要性"
FROM #文献笔记
WHERE contains(年份, "2025")
SORT 重要程度 DESC
```

### 按学科查询文献
```dataview
TABLE 
  title as "标题",
  year as "年份",
  研究质量 as "质量"
FROM #文献笔记
WHERE contains(学科分类, "教育技术学")
SORT 研究质量 DESC
```

### 查询高质量文献
```dataview
TABLE 
  authors as "作者",
  title as "标题",
  year as "年份",
  publication as "期刊"
FROM #文献笔记
WHERE 研究质量 = "⭐⭐⭐⭐⭐"
SORT year DESC
```

### 查询未读文献
```dataview
LIST
FROM #文献笔记
WHERE 阅读状态 = "未读"
SORT 重要程度 DESC
```

## 📈 使用建议

### 标签使用原则
1. **层级清晰**: 一级→二级→三级标签
2. **语义明确**: 标签名称直观易懂
3. **数量适中**: 每个文献3-8个标签
4. **一致性**: 同类内容使用相同标签

### 元数据填写规范
1. **必填字段**: 标题、作者、年份、期刊
2. **选填字段**: DOI、URL、关键词
3. **扩展字段**: 根据需要添加
4. **格式统一**: 日期格式YYYY-MM-DD

### 质量评级标准
1. **客观公正**: 基于文献实际质量
2. **标准一致**: 使用相同评级标准
3. **及时更新**: 随着认识深入调整

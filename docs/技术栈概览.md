# 技术栈概览

## 🏗️ 系统架构

本项目采用**模块化设计**，整合了多个专业工具，构建了完整的毕业论文写作辅助系统。

## 💻 核心技术

| 类别 | 技术 | 版本 | 用途 |
|------|------|------|------|
| **文献管理** | Zotero | 7.0+ | 文献收集、引用生成 |
| **笔记系统** | Obsidian | 1.0+ | 知识管理、双向链接 |
| **后端服务** | Node.js | LTS | MCP服务器、API接口 |
| **脚本开发** | Python | 3.12+ | 自动化工具、配置脚本 |
| **数据存储** | SQLite | - | Zotero数据库 |

## 🔌 关键插件

### Obsidian插件
- **Dataview** - 动态数据查询
- **Templates** - 模板管理
- **Graph View** - 知识图谱可视化

### Zotero插件
- **Zotfile** - PDF注释提取
- **GB/T 7714-2015 CSL** - 标准引用格式
- **茉莉花插件** - 中文文献增强（可选）

## 🎯 技术特点

### 优势
- **标准化**: 采用GB/T 7714-2015引用格式
- **模块化**: 各组件独立，易于维护
- **可视化**: 知识图谱和双向链接
- **自动化**: 脚本化配置和管理

### 适用场景
- 学术论文写作
- 文献综述整理
- 知识管理
- 研究项目管理

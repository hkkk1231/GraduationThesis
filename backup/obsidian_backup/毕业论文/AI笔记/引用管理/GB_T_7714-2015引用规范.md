# GB/T 7714-2015 引用格式模板

## 📋 标准概述

**GB/T 7714-2015《信息与文献 参考文献著录规则》**是中国现行的参考文献著录国家标准，于2015年12月1日实施，替代了GB/T 7714-2005标准。

## 📚 文献类型标识

| 标识 | 文献类型 | 说明 |
|------|----------|------|
| [M] | 专著 | 普通图书、学位论文等 |
| [J] | 期刊文章 | 学术期刊论文 |
| [D] | 学位论文 | 硕士、博士学位论文 |
| [C] | 会议论文 | 学术会议文集 |
| [R] | 报告 | 技术报告、研究报告 |
| [S] | 标准 | 国家标准、行业标准 |
| [P] | 专利 | 专利文献 |
| [N] | 报纸文章 | 报纸发表的文章 |
| [DB/OL] | 联机网络数据库 | 网络数据库 |
| [EB/OL] | 联机网络电子公告 | 网页、网络公告 |

## 🎯 常用引用格式模板

### 1. 期刊论文 [J]
```
[序号] 作者. 题名[J]. 期刊名, 年, 卷(期): 起止页码.
```

**示例**：
```
[1] 李荣琼. 基于人工智能技术的语文课堂教学设计与分析[J]. 未明确期刊, 2025(10): 146-147.
[2] 何龄修. 读南明史[J]. 中国史研究, 1998, 6(3): 167-173.
```

### 2. 专著 [M]
```
[序号] 作者. 书名[M]. 版本项. 出版地: 出版者, 出版年: 引文页码.
```

**示例**：
```
[1] 郑浩峻, 张秀丽. 足式机器人生物控制方法与应用[M]. 北京: 清华大学出版社, 2011.
[2] 汪应洛. 系统工程[M]. 第3版. 北京: 机械工业出版社, 2003: 471-472.
```

### 3. 学位论文 [D]
```
[序号] 作者. 题名[D]. 保存地点: 保存单位, 年份.
```

**示例**：
```
[1] 马欢. 人类活动影响下海河流域典型区水循环变化分析[D]. 北京: 北京大学, 2011.
```

### 4. 会议论文 [C]
```
[序号] 作者. 题名[C]//会议文集名. 出版地: 出版者, 出版年: 起止页码.
```

**示例**：
```
[1] 辛希孟. 信息技术与信息服务国际研讨会会议文集: A集[C]. 北京: 中国社会科学出版社, 1994.
```

### 5. 电子资源 [EB/OL]
```
[序号] 作者. 题名[EB/OL]. (更新或修改日期)[引用日期]. 获取和访问路径.
```

**示例**：
```
[1] 萧钰. 出版业信息化迈入快车道[EB/OL]. (2001-12-19)[2002-04-15]. http://www.reader.com/news/20011219/200112190019.html.
```

### 6. 标准 [S]
```
[序号] 主要责任者. 标准名: 标准编号[S]. 出版地: 出版者, 出版年.
```

**示例**：
```
[1] 全国信息与文献标准化技术委员会. 文献著录: 第四部分 非书资料: GB/T 3792.4—2009[S]. 北京: 中国标准出版社, 2010.
```

## ⚠️ 注意事项

### 标点符号规范
- 使用半角标点符号：`.` `,` `:` `;`
- 标点符号后加空格（句号除外）
- 起止页码使用不间断连字符 `-`

### 作者姓名规范
- 中文作者：姓名全写，如"李荣琼"
- 英文作者：姓全写，名可缩写，如"Smith J"
- 多位作者用逗号分隔，三位以上用"等"

### 其他注意事项
- 期刊名不加书名号
- 出版地必须与出版者对应
- 电子资源需标注引用日期和访问路径
- 文献类型标识根据实际情况选择

## 🔍 引用检查清单

- [ ] 作者姓名格式正确
- [ ] 文献标题完整准确
- [ ] 期刊/书名规范
- [ ] 年、卷、期、页码齐全
- [ ] 标点符号使用半角
- [ ] 文献类型标识正确
- [ ] 电子资源包含引用日期
- [ ] 链接可访问（如适用）

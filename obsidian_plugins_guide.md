# Obsidian插件安装指南

## 必需插件列表

### 1. Zotero Integration
- **功能**: 与Zotero文献管理软件集成
- **安装**: 在Obsidian中设置 → 社区插件 → 浏览 → 搜索"Zotero Integration"
- **配置**:
  - Zotero路径: `C:\Program Files\Zotero\zotero.exe`
  - 文献库路径: `E:\仓库\毕业论文\obsidian\AI笔记\文献笔记`
  - PDF路径: `E:\仓库\毕业论文\obsidian\AI笔记\PDF阅读`

### 2. Dataview
- **功能**: 动态数据查询和展示
- **安装**: 设置 → 社区插件 → 浏览 → 搜索"Dataview"
- **配置**: 启用Dataview查询功能

### 3. Templater
- **功能**: 模板管理和自动化
- **安装**: 设置 → 社区插件 → 浏览 → 搜索"Templater"
- **配置**:
  - 模板文件夹: `E:\仓库\毕业论文\obsidian\AI笔记\模板`
  - 启用自动触发器

### 4. QuickAdd
- **功能**: 快速添加内容和创建笔记
- **安装**: 设置 → 社区插件 → 浏览 → 搜索"QuickAdd"
- **配置**: 设置快速创建文献笔记的选项

## 安装步骤

### 1. 启用社区插件
1. 打开Obsidian
2. 设置 → 第三方插件 → 安全模式 → 关闭
3. 社区插件 → 浏览

### 2. 安装插件
1. 在社区插件浏览中搜索上述插件
2. 点击安装并启用
3. 重启Obsidian

### 3. 配置插件
按照上述配置说明设置每个插件

## 测试安装

### 测试Zotero Integration
1. 在Obsidian中按 `Ctrl+P`
2. 输入"Zotero: Insert literature note"
3. 检查是否能正常工作

### 测试Dataview
1. 创建新笔记
2. 输入示例查询代码
3. 检查是否能正常显示数据

### 测试Templater
1. 创建新笔记
2. 使用模板快捷键
3. 检查模板是否正常应用

## 故障排除

### 常见问题
1. **插件无法安装**: 检查网络连接和Obsidian版本
2. **Zotero连接失败**: 确认Zotero正在运行且路径正确
3. **模板无法加载**: 检查模板文件夹路径

### 解决方案
1. 重启Obsidian
2. 重新安装插件
3. 检查插件权限设置
